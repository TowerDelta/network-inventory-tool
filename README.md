# Network Inventory Tool

Semplice tool CLI in Python per gestire un inventario di dispositivi di rete (IP, hostname, tipo, stato).  
Progetto creato durante lo studio di Networking Basics (Cisco) e Python Essentials.

### Funzionalità
- Aggiungi/rimuovi dispositivi
- Cerca per IP o hostname
- Esporta in CSV (pronto per SAP-like)
- Salvataggio automatico su file

### Tecnologie
Python 3 | CSV | JSON

### Come eseguire
```bash
python main.py
